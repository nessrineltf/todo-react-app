# 📝 Todo App - React

## 📌 Description

This is a simple and responsive **Todo Application** built with React.
The app allows users to manage their daily tasks

---

## 🚀 Features

* ➕ Create new tasks
* ❌ Delete tasks
* 🔍 Filter tasks (Urgente / Moyenne / Basse/ Tous)
* 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

* ⚛️ React
* 🎨 Tailwind CSS
* 🌼 DaisyUI
* 🔍 Lucide Icons

---

## 📷 UI Preview

A clean and minimal design using TailwindCSS and DaisyUI components with modern icons.

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/nessrineltf/todo-react-app.git
```

2. Navigate to the project folder:

```bash
cd todo-react-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```
